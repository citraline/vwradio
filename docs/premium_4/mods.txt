Must have:
x    BUS(to radio)
x    CLK(to radio)
x    DAT(to radio)
x    STB(to radio)
x    BUS(to faceplate)
x    CLK(to faceplate)
x    DAT(to faceplate)
x    STB(to faceplate)
x    LOF(to radio+faceplate), or RST(to radio+faceplate)
x    TAPE CLOCK
x    TAPE DATA
x    TAPE ENABLE
x    TAPE SWITCH

Unsure power, possibly on a separate connector:
    Unswitched 14V?
    Unswitched 5V?
    Main (switched) 5V?
    GND?

Unsure signals:
    Use LOF or RST from radio, or both?
    TAPE FE/ME?    - can be left no-connect
    TAPE MONITOR?  - can be left no-connect
